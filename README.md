## ✨ Key Features

- **🎓 Roles & Access Management:** Full RBAC (Role-Based Access Control) using Spatie Permissions. Distinct portals for Students, Verified Teachers (e.g., Chemistry, Physics), and Admins.
- **📹 Live Video Tutoring:** Real-time, low-latency audio and video communication powered by the **Agora RTC SDK**.
- **📅 Smart Booking System:** localized appointment scheduling enabling students to book subject matter experts efficiently.
- **👑 Advanced Admin Dashboard:** A powerful, deeply customizable administration UI built on **FilamentPHP**, localized completely into Arabic with custom Taj branding (منصة تاج التعليمية).
- **🌍 Full Localization & RTL Support:** Fully localized in Arabic with native Right-to-Left (RTL) interface modeling for the Middle Eastern audience.
- **🔒 Secure API & Authentication:** Token-based security and robust API endpoints handled by **Laravel Sanctum**.

## 🛠️ Technology Stack

The project is structured as a monorepo, distinctively separating the frontend presentation layer from the backend API services.

### Backend (`/backend`)
- **Framework:** Laravel 12.0 (PHP 8.3)
- **Database:** MySQL 8.0
- **Admin Panel:** Filament V3
- **Security:** Laravel Sanctum & Spatie Laravel Permission
- **Testing:** PHPUnit / PestPHP

### Frontend (`/frontend`)
- **Framework:** Next.js 14.2 (App Router / Pages)
- **Language:** TypeScript & React 18
- **Styling:** Tailwind CSS 3.4 & PostCSS
- **Real-time Video:** Agora React UIKit & Agora RTC SDK
- **Testing:** Jest & React Testing Library

## 🚀 Getting Started (Docker Setup)

The easiest way to run the entire Taj Platform stack (Frontend, Backend, and Database) is via **Docker Compose**.

### Prerequisites
- [Docker](https://docs.docker.com/get-docker/) & [Docker Compose](https://docs.docker.com/compose/install/)
- Node.js 20+ (for local frontend development)
- PHP 8.3 & Composer (for local backend development)

### 1. Clone & Prepare
```bash
# Clone the repository
git clone https://github.com/Ammar-1993/Taj-Platform.git
cd Taj-Platform

# Copy Environment Files
cp backend/.env.example backend/.env
# Note: Update Agora credentials and DB settings inside backend/.env if necessary
```

### 2. Start the Docker Environment
Our `docker-compose.yml` is orchestrated to run all services concurrently:
```bash
docker-compose up -d --build
```
*This command spins up:*
- **MySQL Container** (Port `3307` mapped to `3306`)
- **Laravel Backend Engine (PHP 8.3)** (Port `8000`)
- **Next.js Frontend Client** (Port `3000`)

### 3. Application Setup (Backend)
Run these commands inside the `laravel.test` container or locally if you prefer:
```bash
# Enter the Laravel container
docker-compose exec laravel.test bash

# Install dependencies and setup Laravel
composer install
php artisan key:generate
php artisan migrate --seed # Seeds the database and verified teacher accounts
```

#### Endpoints
- **Frontend App:** `http://localhost:3000`
- **Backend API:** `http://localhost:8000/api`
- **Filament Admin Panel:** `http://localhost:8000/admin` (or configured route)

## 🏗️ Architecture & Workflows

1. **Routing:** The Next.js frontend interacts directly with Laravel endpoints via `axios`.
2. **Video Workflow:** When a session is initialized, Next.js calls the Agora API wrapper endpoints to retrieve connection tokens securely generated by the Laravel Backend.
3. **Admin Actions:** Administrators manage the syllabus, verifications, bookings, and user statuses via the Filament Admin panel seamlessly integrated within Laravel.
